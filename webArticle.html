PROJECT

World Xplore

CATEGORY
Web Development
LANGUAGE
HTML, CSS, Bootstrap, jQuery
HTML5 CSS3 Bootstrap jQuery Flexbox

The goal of this project is to create a responsive home page design for travel firm World Xplore. The logo, navigation, background video, article section, and footer are the primary needs. Using CSS media queries and Flexbox, the website is also completely responsive.

The only frameworks used to create the website are HTML, CSS, and JavaScript.

Key Libraries Used

HTML5
Semantic markup for structured content and improved accessibility
CSS3 + Flexbox
Modern styling with Flexbox layouts and CSS media queries for responsiveness
Bootstrap
Grid system and components for rapid, responsive development
