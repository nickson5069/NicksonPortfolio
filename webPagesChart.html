Bar Charts Line Charts Pie Charts Scatter Plots

This project aims to develop several types of charts using full-page JavaScript. To display static data, it is necessary to develop a variety of charts. The website is fully responsive and uses Flexbox and CSS media queries to achieve this.

HTML, CSS, FullPage JS, and Chart JS are used to develop the framework. Each type of graphic will show the appropriate data.

Key Libraries Used

FullPage.js
Creates smooth, full-screen scrolling sections with navigation dots and keyboard support
Chart.js
Renders beautiful, animated charts with multiple types: bar, line, pie, and radar
Bootstrap + Flexbox
Ensures responsive design that works perfectly on all screen sizes
